Entorno para terminales de texto. Permite cambiar colores, tipo de letra, transparencias, imagen de fondo... Descarga de <http://sourceforge.net/projects/bashish/files/> . Se instala con el cl치sico:

	$ ./configure && make && sudo make install

>$ bashish list (Muestra el listado de temas)

>$ bashish fire (Abre la terminal col el tema fire)

>$ BASHISH_DISABLED=1 un_comando (Lanza el comando que especifiquemos deshabilitando bashish)
