Controla lxpanel.

>$ lxpanelctl menu (abre el menú de lxpanel)

>$ lxpanelctl config (Su configuración)

>$ lxpanelctl run (Cuadro para entrar una aplicación para ejecutarla)

Otras opciones:

restart (Reiniciarlo)

exit (Matarlo)
